# My Portfolio

Đây là mã nguồn website portfolio cá nhân của tôi.

## Demo
Bạn có thể xem trực tiếp tại: [https://GaHongHacDev.github.io/MyPortfolio/](https://GaHongHacDev.github.io/MyPortfolio/)

## Giới thiệu
- Website này giới thiệu về tôi, các kỹ năng, dự án, và kinh nghiệm cá nhân.
- Viết bằng HTML, CSS, JavaScript, Bootstrap.
- Responsive, hiển thị tốt trên mọi thiết bị.

## Cách sử dụng
1. Clone repo về máy:
git clone https://github.com/GaHongHacDev/MyPortfolio.git

2. Mở file `index.html` bằng trình duyệt, hoặc deploy lên GitHub Pages.
## Tính năng chính
- Trang giới thiệu bản thân.
- Danh sách kỹ năng và công nghệ sử dụng.
- Portfolio dự án.
- Thông tin liên hệ.

## Đóng góp
- Repo này là mã nguồn mở. Nếu bạn muốn đóng góp hoặc chỉnh sửa, hãy fork repo và gửi pull request.
